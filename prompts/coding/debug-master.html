Pro Tips:

  • Include the FULL error message - every character matters
  • Provide 10-20 lines of context around the error, not just the error line
  • Mention your environment (Node version, browser, OS) for environment-specific bugs
  • If it's intermittent, note when it works vs when it fails
  • Include any recent changes you made before the bug appeared

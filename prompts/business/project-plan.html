Project Planning Template

Turn overwhelming projects into step-by-step action plans that actually get completed on time.

Example Use Case

Project: Launch new company website in 60 days

Key Insight: "Critical path: Content creation (20 days) → Design (15 days) → Development (15 days) → Testing (5 days) → Launch. Start content NOW or everything delays."

Pro Tips

  • Add 20% buffer time to all estimates
  • Identify critical path - what blocks everything else
  • Get commitments in writing from team members
  • Plan for people being sick/unavailable
  • Define "done" clearly for each task
  • Update the plan weekly - it's a living document
